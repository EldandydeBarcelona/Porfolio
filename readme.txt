Este es el portfolio de Pablo Álvarez Fernández,
esta basado en unas paginas web realizadas durante
el curso.

Para hacerlo he divido el html y el css, en el 
html lo he divido en un header y en tres secciones.

En el header he creado un navbar para poner el logo
de AFA(ya que es un trabajo) y el hero, luego tenemos
un div con un contenedor dentro para poner el titulo
y un boton para ir a la parte "sobremi".

En sobremi tenemos todos mis proyectos y cosas sobremi,
dentro tengo un contenedor solo con el titulo, otro 
div dentro de otro div.

Una seccion para cuadrar bien la imagen.

Y por ultimo un contacto con su info.

Para terminar he decorado el header con una imagen 
de fondo, la pagina la cambie a color negro y he 
utilizado flexbox para cuadrarlo mejor.
